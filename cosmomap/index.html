COSMOMAP 2¶

COSmic Microwave Operators MAp-making Preconditioner is a package which performs iteratively the solution to the CMB map-making problem via preconditioning the system-matrix with Deflation preconditioner ( see Szyldlarski et al.2014). The whole system matrices are implemented as Linear Operators. Visit the Github repository COSMOMAP2.
