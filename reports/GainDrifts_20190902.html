1. Summary¶

09/02/2019

In my previous posting, I've presented the parametrization adopted to inject (and mitigate) gain drift systematics.

In this posting I show that we are able to track $\sim10$ percent gain fluctuations on the time scale of up to 1 minute by regressing the time samples (including noise and sky signal) with CMB Temperature template map. Moreover, smaller gain fluctuations (order of few per cents) are harder to track because noise affects the estimates, in this case a longer time scale might be helpful .

Simulation setups¶

Toast Params ( values from v0 Litebird-IMO wiki page:

  • Detector pairs: 2
  • frequency band: 89 GHz
  • T_obs : 5 hours
  • sampling freq: 20 Hz
  • precession period :192.348 min
  • spin period: 20 min
  • Noise simulated : white
  • NET : 30 uK sqrt(s)
  • HWP : False
  • spin angle : 50
  • precession angle :45
  • MADAM destriper (baseline=60 sec)
  • Input maps: Monopole, Dipole, CMB T , Galactic emission @90 GHz

Gain drift sim¶

Simulate per detector a time stream signal $g(t)$ fluctuating around 1 with a low frequency component with a 1/f dependence $fknee = 150 $ mHz .

Multiply $g$ by the detector TOD.

Gain estimation¶

The procedure to mitigate gain mismatches is performed on each chunck (related to $\tau$) of time samples $d_{t , t+\tau}$ of a given detector:

  1. Apply a high-pass filter to $d_{t , t+\tau} $ with cutoff frequency $f_{c}=5$ mHz
  2. Regress $d_{t , t+\tau} $ against signal-only template $T$ and estimate the regression slope $\hat{g}$ as $$ \hat{g} = \frac { T^{t} d } { T^{t} T } $$
  3. Mitigate the leakage, dividing $d_{t , t+\tau} $ by $\hat{g}$

from scipy.signal import butter, lfilter

def butter_bandpass(lowcut, fs, order=5):
    nyq = 0.5 * fs
    low = lowcut / nyq
    b, a = butter(order, low, btype='highpass')
    return b, a


def butter_bandpass_filter( lowcut,   fs, data, order=5):
    b, a = butter_bandpass(lowcut,  fs, order=order)
    padl  = np.int_( fs/(lowcut)) 
    y = signal.filtfilt (b, a, data , padlen= padl, padtype='constant')
    return y


def band_pass_filter(fcutoff, fsampl, data, filtermode='high' ):
    alpha=2. *fcutoff  /fsampl
    padlength = np.int_( fsampl/(fcutoff)) 
    b, a = signal.butter(4, alpha , filtermode)
    pdata=pl.pad(data ,  pad_width =(padlength,padlength  ), mode ='symmetric', reflect_type='odd' )
    filtsignal=  signal.filtfilt(b, a, pdata,  padlen=0 )
    #return filtsignal
    return filtsignal[slice ( padlength ,  -padlength) ]

def bin_gains(tau, fs,obs,  g) : 
    sizech = pl.int_( tau *fs)  ; nch =pl.int_(  obs *fs*3600  /sizech) 
    gb = pl.zeros(nch)
    for i in range (nch): 
        gb[i] =g[slice(i*sizech, (i+1)*sizech)] .mean()
    return gb
