Simulating CO T->P leakage from bandpass mismatch¶

We injected bandpass mismatches by simulating 1 yr of LB signal -only observations performed by 2 detector pair. We model the observed TOD signal as of the two detectors A, B as :

$$ d^A_t = f s_{CO} $$ $$ d^B_t = \epsilon f s_{CO} $$

where $f$ and $\epsilon $ parametrize the bandpass mismatch of the detector pair and $s_{CO} $ is the scanned CO intensity only map, $m_{CO}$ ( no polarization, $Q=U=0$) with a pointing matrix $A$ encoding the LB scanning strategy, i.e. : $$ s_{CO} = A m_{CO} $$

If we consider pair sum and pair differences : $$ d^+ = \frac{d^A +d^B}{2} = \frac{f(1+\epsilon)} {2} s_{CO} $$

$$ d^- = \frac{d^A -d^B}{2} = \frac{f(1-\epsilon)} {2} s_{CO} $$

We can estimate $m_{leak} = (Q, U)_{leak}$ maps from $d^-$ timestream as : $$ m_{leak} = (A^t A)^{-1} A^t d^-= (A^t A)^{-1} A^t \frac{f(1-\epsilon)} {2} s_{CO} $$ for an optimized scanning strategy we can approximate : $$ (A^t A)^{-1} \approx \begin{bmatrix} 1 & 0 & 0 \\ 0& 2 & 0 \\ 0 & 0 & 2 \end{bmatrix} $$ so that: $$ m_{leak} = f(1-\epsilon ) A^t s_{CO}=f(1-\epsilon ) A^t A\, m_{CO} $$

Getting a consistent notation with Tommaso¶

In these slides presented by Tommaso, he uses a slightly different notation as compared to the one above. Notice that we get the same result if :

$$\Delta_{Tom} \equiv f (1- \epsilon)$$

In the following simulations I've run TOAST on a configuration similar to the Tommaso's one:

  • nside=64, $Ndet=114$, $\Delta=7.5$ %
  • one on hand we want to check whether the cross-link approximation (rescaling with $\sqrt{N_{det}$) is correct
  • on the other hand, this will provide consistency checks between the two approaches

I can run bandpass mismatch sims with two different option:

  1. same for all detector pairs,i.e. $f= 0.5$ and $\epsilon=0.85 $, so that $\Delta=7.5 \% $
  2. random mismatch per detector pair, i.e. $f$ uniform between 0 and 1, $\epsilon $ s.t. $f (1- \epsilon)=7.5 \% $
