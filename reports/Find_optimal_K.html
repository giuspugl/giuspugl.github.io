Defining patches via clustering methodologies¶

2019-09-13
Giuseppe

  1. Consider maps of $\beta_d$ and the related uncertainties $\sigma(\beta_d)$ from Planck Legacy Archive
  2. From the maps of $\beta_d$ and $\sigma(\beta_d)$ estimate for each pixel the continuous probability distribution $\mathcal{P}$ assumed to be normal distribution $ \mathcal{P} \sim N( {\beta }_d ,\sigma(\beta_d))$
  3. Estimate the Hellinger distance which is used to quantify the affinity between two probability distributions
  4. Look for $K$ patches given the affinity matrix with Hierarchical clustering method.

Overlapping area: $$A_{ij} \equiv \int_X dx \sqrt{\mathcal{P}_i \mathcal{P}_j } $$ Hellinger distance: $$H_{ij} = \sqrt{1-A_{ij} }$$

What is the optimal number of clusters ?¶

There is a trade off between the intra and inter-cluster distances. The former tends to increase with $K$ and it's defined by the minimum of the distance between cluster centroids. The latter is a quantity related to the sizes of clusters and how many elements belong to each cluster. Hence, intuitively, tends to decrease with $K$.

  • similarity including spatial location (Haversine) + (Hellinger)

def plotclusters(labels,imap):
    outm= pl.zeros_like(imap)
    for i in range(labels.max()+1):
        pixs=pl.where(labels==i)
        outm[pixs]=imap[pixs].mean()

    return outm

def intracluster_distance(K,X, labels  ):
    mu = pl.zeros( K*X.shape[1]).reshape(K,X.shape[1])
    MD = pl.zeros( K)
    for k in range(K):
        ck =pl.where(labels == k )[0]
        Xk =X[ck,:  ]
        mu[k] = Xk.mean(axis=0)
        Nk = len(ck)
        MD[k] = pl.linalg.norm(Xk-mu[k],axis=1).sum()/Nk
    return mu, MD

def hellinger_distance(x,y ) :
    #estimating hellinger distance from https://en.wikipedia.org/wiki/Hellinger_distance
    mu1= x[0]; sigma1=x[1];
    mu2= y[0]; sigma2=y[1] ; 
    

    BC = (pl.sqrt(2. *sigma1*sigma2/(sigma1**2 +sigma2**2))
          * pl.exp(-1/4. *(mu1-mu2)**2/(sigma1**2 +sigma2**2)))

    return pl.sqrt(1- BC ) 

def setup_features(X, nside , affinity='mult'  ) :
    Npix=hp.nside2npix ( nside )
    #prepare features  Coordinates
    longi, lat= pl.deg2rad( hp.pix2ang(nside=nside ,ipix=pl.arange(Npix), lonlat=True )  )
    mask =  pl.ma.masked_inside(longi, 0 , pl.pi ).mask
    longi [  mask] =  - longi [mask]
    longi [  pl.logical_not(mask) ] =  2* pl.pi - longi [pl.logical_not ( mask) ]
    Theta= pl.array([ lat ,  longi ]).T
    #create feature vector

    Xtilde = pl.concatenate( [X , Theta ], axis=1)
    # apply MinMaxScaler
    min_max_scaler = preprocessing.MinMaxScaler()

    #Xtilde= min_max_scaler.fit_transform(Xtilde)
    # prepare Affinity  matrix =  Euclidean x Haversine distances
    haversine = DistanceMetric.get_metric('haversine')
    if nside==32:
        paramdistance_matr=pl.load('/Users/peppe/work/heavy_maps/affinity_hellinger_nside32_betadust.npy')
    else:    
        hellinger = DistanceMetric.get_metric( hellinger_distance) 
        
        paramdistance_matr=  hellinger.pairwise( Xuncertainties )
        
    if affinity =='mult':
        angdist_matr =haversine.pairwise( Theta  )
        angdist_matr =pl.ma.fix_invalid(angdist_matr,  fill_value=pl.pi).data
        #paramdistance_matr= pairwise_distances( X, metric='euclidean')
        angdist_matr/=angdist_matr.max() 
        tot=pl.multiply ( paramdistance_matr, angdist_matr)

        Xtilde = X
