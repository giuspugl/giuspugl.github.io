Mueller Matrix¶

A general definition can be given by $$ M_{ij} =\frac{1}{2} Tr (\sigma_i J \sigma_j J^{+}) $$

Rotation matrix¶

each angle involved in the computation of Mueller Matrices has to be taken into account with appropriate rotation matrices, in the following we consider

  • $\xi $ the detector orientation
  • $\psi$ the instrument orientation
  • $\theta_{HWP}$ the HWP angle

M = (Mpol* Mxi* Mtheta.T*M_HWP*Mtheta*Mpsi)

The total Muller matrix is defined as :

$M = M_{pol} M_{\xi} M_{\theta} ^T M_{HWP} M_{\theta} M_{\psi} $

If we define $\epsilon= \delta^2/\eta^2$ and factorize out $\eta^2 *(1+\epsilon)/2\equiv C $ and work out a bit the trigonometry we get :

  • $ M_{II} = C \left[ T + \frac{(1-\epsilon)}{(1+\epsilon)} \rho \cos (2\theta) \right] $

  • $ M_{IQ} = C \left[ \rho \cos(2\theta +2 \psi ) + \frac{(1-\epsilon)}{(1+\epsilon)} \left(\frac{(T+c)}{2} \cos 2\psi + \frac{(T-c)}{2} \cos ( 2\psi +4\theta) \right) \right] $

  • $ M_{IU} = C \left[ \rho \sin(2\theta +2 \psi ) + \frac{(1-\epsilon)}{(1+\epsilon)} \left(\frac{(T+c)}{2} \sin 2\psi + \frac{(T-c)}{2} \sin ( 2\psi +4\theta) \right) \right] $

notice that for the cases shown above (ideal HWP and no-HWP ) the expressions above simplify as expected.

These weights are implemented in toast in a new operator OpMuellerPointingHpix in the mueller_matrix branch.
